# sarriaxd.github.io

Personal homepage for [Qi Wang](https://sarriaxd.github.io) — software engineer based in Canada.

Built quietly with [Astro](https://astro.build), [Tailwind CSS](https://tailwindcss.com), and a small amount of vanilla TypeScript for restrained, accessible motion.

## Develop

```bash
npm install
npm run dev
```

Site runs at `http://localhost:4321`.

## Build

```bash
npm run build
npm run preview
```

## Deploy

GitHub Actions builds and deploys to GitHub Pages on every push to `main`. The deployed site is served at the root of `https://sarriaxd.github.io`.
